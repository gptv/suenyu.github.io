教育点

偶尔听到初中部眼保健操的音乐，翻检看到2012年眼保健操利弊的讨论，比较明确的结论是，60年前推行时，并无科学依据，反倒类似聊胜于无所以推行，之后形成惯例，漫说动不得，能追问这玩意有没有用的已然是少数。以举国如此的效果，以医学这种显然需要实证的学科，决策的过程，无疑早已寻不到，这背后，无非不扎实不认真，教育部是团队还是团伙呢？
高考还有几天，自恢复以来，命题流程官方解析几乎无一可查；对比台湾指考每年公开出的命题相关会议纪要详细解析等，大陆命题组只怕连团伙都还不是。
把更多应该被记忆的事情公示出来，是非对错公私利弊，一时即便不清，至少，也给之后一个起点。
翻看手机中若干退过多轮的学校事务微信群，就可以知道，即便北大附中，这个记录与公示的路，也还太长。
技术问题吗？显然不是。

New York school district’s facial recognition system sparks privacy fears，这个标题显然不如
The First Public Schools In The US Will Start Using Facial Recognition Next Week，项目上周四被叫停，也并无意外。
和国内课堂内人脸识别虽然并非一事，但人脸识别技术进学校，大势所趋，只在早晚深浅。
对标北大，附中出入门的人脸识别正在部署；课堂如何？继续等。

学院事

每周一例行书院督导会，提及优秀书院评选，发现一直没有一个标准的文本，无标准，实际也就无建设；课程如此，书院也是。继续等。
靠书院督导去主持书院工作规划和议事会方案，能完成书院建设吗？能；能推进书院自治吗？不能。
每个书院一个Teams和一个Github，需要多长时间？

这周二集团会，早九点到晚九点，完成学生工作小组的汇报，完成学院框架汇报，完成荣誉文凭方案汇报。
定位明白，有所为有所不为，事情也就可以清晰了。

周三带subit到微软中国，附中信息化建设，不乏无可奈何，但这个社团，正在去往成为一个传说的路上。
看很多学校纠结在信息化的点，但真正死于推进路上的其实不多；更多的学校其实死在寻找的路上，找不到或根本看不懂。O365或其他，一直摆在网上，部署的成本可以是北大附中的零成本，也可以是有些学校的几百万。信息不对称，在技术领域，要怪的，大多只是自己。

周四周五做研究性学习与项目学习关系梳理，探底结论：项目式学习其实是研究性学习被学术疲劳之后，一个学科内容与学习方式解弊的更新概念。在这个意义上，基于2000年初国家推行研学初心，看北大附中研学历史中语文小论文写作、数学建模等学科研学，至少当时，即便北大附中，肯定是没看懂研学的。所以，本质上，推项目学习和重启研学是一件事，但世事总要迁延，随之更新吧。20年前推行研学的人，看今日中学研学局，或者也是如鱼饮水吧。
